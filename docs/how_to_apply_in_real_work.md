# How to Apply This Repo in Real Work

## Scenario 1: HR asks whether termination is safe

### Input
HR แจ้งว่า ลูกจ้างขาดงาน 3 วัน แต่มีวันหยุดคั่นกลาง และต้องการทราบว่าสามารถเลิกจ้างโดยไม่จ่ายค่าชดเชยได้หรือไม่

### Agent process
1. Classify issue: Abandonment of duty / Severance pay
2. Generate keywords:
   - ละทิ้งหน้าที่ สามวันทำงานติดต่อกัน
   - ขาดงาน วันหยุดคั่น
   - ไม่มีเหตุอันสมควร
   - มาตรา 119 (5)
3. Search official/reliable sources
4. Record search log
5. Extract relevant cases
6. Compare facts
7. Prepare memo
8. Require legal review before action

### Output
- Case table
- Risk analysis
- Evidence needed
- Recommended HR action

## Scenario 2: Audit asks whether OT practice creates risk

### Input
Audit พบว่าพนักงานเริ่มงานก่อนเวลา 30 นาทีเป็นประจำ แต่ระบบไม่แสดง OT ก่อนเวลา

### Agent process
1. Classify issue: OT / working time / employer instruction
2. Generate keywords:
   - ค่าล่วงเวลา ก่อนเวลาทำงาน
   - นายจ้างสั่งให้ทำงานล่วงเวลา
   - ทำงานล่วงเวลาโดยปริยาย
3. Search precedents
4. Map evidence required:
   - Time record
   - Supervisor instruction
   - CCTV / access log
   - Work assignment
5. Prepare audit recommendation

## Scenario 3: Legal memo for management

### Output structure
- Facts
- Legal issue
- Case trend
- Risk level
- Recommended action
- Evidence to preserve
- Human review required
