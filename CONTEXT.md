# CONTEXT.md

## Human-authored project context

โครงการนี้ต้องการสร้าง AI Agent เพื่อช่วยค้นหาและวิเคราะห์แนวคำพิพากษาศาลฎีกาประเภทคดีแรงงาน โดยผู้ใช้หลักคือ HR, HRBP, Labour Compliance, Internal Auditor, Legal Researcher และผู้บริหารที่ต้องการเข้าใจความเสี่ยงด้านแรงงานจากข้อเท็จจริงที่เกิดขึ้นจริงในองค์กร

## Why this project exists

งานค้นคำพิพากษาศาลฎีกาด้านแรงงานมีความยากเพราะคำค้นมักไม่ตรงกับภาษาที่ผู้ใช้งานทั่วไปใช้ เช่น ผู้ใช้พูดว่า “ขาดงานหลายวัน” แต่แนวคำพิพากษาอาจใช้คำว่า “ละทิ้งหน้าที่เป็นเวลาสามวันทำงานติดต่อกันโดยไม่มีเหตุอันสมควร” ดังนั้น Agent ต้องช่วยแปลงภาษาคนทำงานให้เป็นภาษากฎหมายที่ค้นได้

## Domain scope

Agent นี้เน้นประเด็นคดีแรงงาน เช่น:
- เลิกจ้างไม่เป็นธรรม
- ค่าชดเชย
- สินจ้างแทนการบอกกล่าวล่วงหน้า
- ค่าจ้าง ค่าล่วงเวลา ค่าทำงานในวันหยุด
- การลงโทษทางวินัย
- หนังสือเตือน
- ละทิ้งหน้าที่ / ขาดงาน
- สัญญาจ้างแรงงาน
- ข้อตกลงเกี่ยวกับสภาพการจ้าง
- แรงงานสัมพันธ์
- ประกันสังคม / เงินทดแทน
- การจ้างเหมาช่วง / outsource / subcontractor

## Expected reasoning style

Agent ต้องคิดแบบนักกฎหมายแรงงานและผู้ตรวจประเมิน:

1. แยกข้อเท็จจริงที่ได้รับ
2. ระบุประเด็นกฎหมาย
3. สร้างคำค้นที่เหมาะสม
4. ค้นจากแหล่งที่น่าเชื่อถือ
5. บันทึกแหล่งข้อมูลและ search log
6. สรุปคำพิพากษาโดยไม่บิดเบือน
7. เปรียบเทียบความเหมือนและความต่างของข้อเท็จจริง
8. สรุปเป็นข้อเสนอแนะเชิง HR / Compliance / Audit
9. ระบุข้อจำกัดทุกครั้ง

## Success definition

ผลลัพธ์ที่ดีต้อง:
- ไม่สร้างเลขฎีกาเอง
- ไม่อ้างคำพิพากษาโดยไม่มีแหล่งที่มา
- แยกข้อเท็จจริง ข้อกฎหมาย และความเห็นอย่างชัดเจน
- มีคำค้นและ search log ตรวจสอบย้อนกลับได้
- มี human review สำหรับเรื่องเสี่ยงสูง
