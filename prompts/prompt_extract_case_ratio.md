# Prompt: Extract Case Ratio

อ่านคำพิพากษาหรือสรุปคำพิพากษาที่ให้มา แล้วแยกข้อมูลดังนี้:

1. เลขคำพิพากษา
2. ปี
3. ข้อเท็จจริงหลัก
4. ประเด็นที่ศาลวินิจฉัย
5. กฎหมายที่เกี่ยวข้อง
6. เหตุผลของศาล
7. ผลคำพิพากษา
8. หลักกฎหมายที่ได้
9. ข้อจำกัดในการนำไปใช้

ให้เขียนแบบสรุป ไม่คัดลอกข้อความยาว และแยก court reasoning ออกจากความเห็นของผู้วิเคราะห์
