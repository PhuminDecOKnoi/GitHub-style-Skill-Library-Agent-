# User Issue Sample

## Issue
ลูกจ้างขาดงาน 3 วัน โดยมีวันหยุดคั่นกลาง นายจ้างต้องการทราบว่าสามารถเลิกจ้างโดยไม่จ่ายค่าชดเชยได้หรือไม่

## Known facts
- ลูกจ้างไม่มาทำงานวันที่ 1, 2 และ 4
- วันที่ 3 เป็นวันหยุดประจำสัปดาห์
- ยังไม่ทราบว่าลูกจ้างมีเหตุอันสมควรหรือไม่
- ยังไม่ทราบว่ามีการติดต่อแจ้งเหตุหรือไม่

## Missing facts
- ตารางทำงานจริง
- ข้อบังคับเกี่ยวกับการทำงาน
- หลักฐานการติดต่อ
- หนังสือเตือนหรือประวัติวินัย
- เหตุผลของลูกจ้าง
