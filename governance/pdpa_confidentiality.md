# PDPA and Confidentiality

## Principles
- ใช้ข้อมูลส่วนบุคคลเท่าที่จำเป็น
- หลีกเลี่ยงชื่อจริง เลขบัตรประชาชน เบอร์โทร ที่อยู่ รายละเอียดสุขภาพ หรือข้อมูลอ่อนไหว หากไม่จำเป็นต่อประเด็นกฎหมาย
- ใช้รหัสแทนชื่อ เช่น Employee A, Employer B, Site C
- ไม่ใส่ข้อมูลส่วนบุคคลใน public repository

## Recommended practice
- เก็บไฟล์จริงใน private storage
- เก็บเฉพาะ summary / anonymized facts ใน repo
- จำกัดสิทธิ์เข้าถึง evidence folder
