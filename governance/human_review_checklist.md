# Human Review Checklist

ใช้ตรวจทานก่อนนำผลลัพธ์ไปใช้จริง

| Check | Yes/No | Note |
|---|---|---|
| ระบุข้อเท็จจริงครบหรือไม่ |  |  |
| มีข้อเท็จจริงสำคัญที่ยังขาดหรือไม่ |  |  |
| คำพิพากษาที่อ้างมีแหล่งตรวจสอบได้หรือไม่ |  |  |
| วิเคราะห์ตรงกับข้อเท็จจริงหรือไม่ |  |  |
| แยกข้อกฎหมายกับความเห็นชัดเจนหรือไม่ |  |  |
| มีความเสี่ยงจากการนำไปใช้ผิดบริบทหรือไม่ |  |  |
| จำเป็นต้องให้ทนาย/ฝ่ายกฎหมายตรวจหรือไม่ |  |  |
| มีการปกปิดข้อมูลส่วนบุคคลที่ไม่จำเป็นหรือไม่ |  |  |
