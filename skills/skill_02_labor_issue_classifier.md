# Skill 02: Labour Issue Classifier

## Purpose
จัดหมวดหมู่ประเด็นคดีแรงงานก่อนค้นคำพิพากษา

## Categories

| Category | Thai issue | Typical keywords |
|---|---|---|
| Termination | เลิกจ้าง | เลิกจ้าง, บอกเลิกสัญญาจ้าง, ให้ออก |
| Severance Pay | ค่าชดเชย | ค่าชดเชย, มาตรา 118, มาตรา 119 |
| Unfair Dismissal | เลิกจ้างไม่เป็นธรรม | ไม่เป็นธรรม, ศาลแรงงาน, ค่าเสียหาย |
| Notice Pay | สินจ้างแทนการบอกกล่าวล่วงหน้า | บอกกล่าวล่วงหน้า, สินจ้างแทน |
| Wages | ค่าจ้าง | ค่าจ้าง, เงินเดือน, ประโยชน์ตอบแทน |
| OT/Holiday Pay | OT / วันหยุด | ค่าล่วงเวลา, วันหยุด, ทำงานในวันหยุด |
| Discipline | วินัย | หนังสือเตือน, ฝ่าฝืนข้อบังคับ, ความผิดร้ายแรง |
| Abandonment | ละทิ้งหน้าที่ | ขาดงาน, สามวันทำงาน, ไม่มีเหตุอันสมควร |
| Employment Contract | สัญญาจ้าง | สัญญาจ้างแรงงาน, ทดลองงาน, fixed term |
| Working Conditions | สภาพการจ้าง | ข้อตกลงเกี่ยวกับสภาพการจ้าง, เปลี่ยนแปลงสภาพการจ้าง |
| Labour Relations | แรงงานสัมพันธ์ | สหภาพ, ข้อเรียกร้อง, lockout, strike |
| Social Security | ประกันสังคม | ผู้ประกันตน, นายจ้าง, เงินสมทบ |
| Workmen Compensation | เงินทดแทน | ประสบอันตราย, เจ็บป่วยจากงาน |
| Outsourcing | จ้างเหมา | ผู้รับเหมาช่วง, นายจ้างตัวจริง, ความสัมพันธ์การจ้าง |

```markdown
- Sub-category:
- Relevant law:
- Key facts:
- Search terms:
- Risk level:
```
