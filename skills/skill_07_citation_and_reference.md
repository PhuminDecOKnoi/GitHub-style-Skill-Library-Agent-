# Skill 07: Citation and Reference

## Purpose
กำหนดรูปแบบการอ้างอิงคำพิพากษาและแหล่งข้อมูล

## Minimum citation elements
- Court
- Case number
- Year
- Source name
- URL or reference path
- Date accessed
- Relevant issue

## Thai legal citation format
```text
คำพิพากษาศาลฎีกาที่ xxxx/yyyy
```

## Rules
- Do not cite unverified case numbers.
- Do not treat summaries from secondary sources as primary authority.
- If only secondary source is available, label it clearly.
- Keep a record in search log.
